i. var: The scope of a var variable is functional scope. It can be updated and re-declared into the scope. It can be declared without initialization.
ii. let: It can be declared without initialization. It can be updated but cannot be re-declared into the scope. It can be declared without initialization.
iii. const: The scope of a const variable is block scope. It cannot be updated or re-declared into the scope. It cannot be declared without initialization.

Regular functions created using function declarations or expressions are constructible and callable. Since regular functions are constructible, they can be called using the new keyword. However, the arrow functions are only callable and not constructible, i.e arrow functions can never be used as constructor functions.
